# ประกาศกระทรวงอุตสาหกรรม ฉบับที่ 8 (พ.ศ. 2566) ออกตามความในพระราชบัญญัติมาตรฐานผลิตภัณฑ์อุตสาหกรรม   พ.ศ. 2511 เรื่อง  กำหนดอัตราค่าใช้จ่ายในการตรวจสอบผลิตภัณฑ์อุตสาหกรรม ผ้าสีกากีสำหรับเครื่องแบบข้าราชการตำรวจ : ผ้าประเภทสำนักงาน

ประกาศกระทรวงอุตสาหกรรม ฉบับที่ 8 (พ.ศ. 256 6) ออกตามความในพระราชบัญญัติมาตรฐานผลิตภัณฑ์อุตสาหกรรม พ.ศ. 2511 เรื่อง กำหนดอัตราค่าใช้จ่ายในการตรวจสอบผลิตภัณฑ์อุตสาหกรรม ผ้าสีกากีสำหรับเครื่องแบบข้าราชการตำรวจ : ผ้าประเภทสานักงาน อาศัยอานาจตามความในมาตรา 6 แห่งพระราชบัญญัติมาตรฐานผลิตภัณฑ์อุตสาหกรรม พ.ศ. 2511 ซึ่งแก้ไขเพิ่มเติมโดยพระราชบัญญัติมาตรฐานผลิตภัณฑ์อุตสาหกรรม (ฉบับที่ 6) พ.ศ. 2548 รัฐมนตรีว่าการกระทรวงอุตสาหกรรมออกประกาศกาหนดอัตราค่าใช้จ่ายในการตรวจสอบ ผลิตภัณฑ์อุตสาหกรรมตามมาตรา 16 มาตรา 20 มาตรา 20 ทวิ มาตรา 21 มาตรา 21 ทวิ และมาตรา 44 (1) เฉพาะผลิตภัณฑ์อุตสาหกรรมที่ใช้เครื่องหมายมาตรฐานตามมาตรา 16 หรือที่ได้รับ ใ บอนุญาตหรือได้รับอนุญาตตามมาตรา 20 มาตรา 20 ทวิ มาตรา 21 หรือมาตรา 21 ทวิ ไว้ ดังต่อไปนี้ มาตรฐาน ผลิตภัณฑ์ อุตสาหกรรม เลขที่ มาตรฐาน รายละเอียด อัตราค่าใช้จ่าย ต่อตัวอย่าง 1 ชุด (บาท) หมายเหตุ ผ้าสีกากี สำหรับ เครื่องแบบ ข้าราชการ ตำรวจ : ผ้าประเภท สำนักงาน มอก. 2686 - 2558 ความกว้างของผ้า 200 มวลของผ้าต่อพื้นที่ 1 m 2 290 วัสดุและ การทำ จำนวน เส้นด้าย แนวเส้นด้ายยืน 68 ± 4 เส้น ต่อ 2.54 cm 340 แนวเส้นด้ายพุ่ง 52 ± 3 เส้น ต่อ 2.54 cm ขนาด เส้นด้าย เส้นด้ายยืน 14 ± 3 ( English cotton count ) 660 เส้นด้ายพุ่ง 14 ± 3 ( English cotton count ) ้ หนา 44 ่ เลม 140 ตอนพิเศษ 183 ง ราชกิจจานุเบกษา 27 กรกฎาคม 2566

มาตรฐาน ผลิตภัณฑ์ อุตสาหกรรม เลขที่ มาตรฐาน รายละเอียด อัตราค่าใช้จ่าย ต่อตัวอย่าง 1 ชุด (บาท) หมายเหตุ เส้นด้ายยืนและเส้นด้ายพุ่ง ต้องตีเกลียวสองเส้น 300 ส่วนผสมเส้นใยเป็นเส้นใย พอลิเอสเทอร์ ( 80 ± 3 ) % และเส้นใยเรยอน ( 2 0 ± 3 ) % 630 ลักษณะการทอ 300 คุณลักษณะ ที่ต้องการ ลักษณะทั่วไป 300 ความนุ่ม 1,000 สี 750 คุณลักษณะ ทางกายภาพ การเปลี่ยนแปลง ขนาดภายหลัง การซักและ ทำให้แห้ง 3 รอบ 900 ความคงทนของสี ต่อการซัก 350 ความคงทนของสี ต่อเหงื่อ 410 ความคงทนของสี ต่อการขัดถู 270 ความคงทนของสี ต่อแสงซีนอนอาร์ก เทียบกับผ้า บลูวูลมาตรฐาน 850 ความคงทนของสี ต่อการกดทับด้วย ความร้อน 390 ความคงทนต่อ การขึ้นเม็ดภายหลัง การขัดถู (7 000 รอบ) 540 ้ หนา 45 ่ เลม 140 ตอนพิเศษ 183 ง ราชกิจจานุเบกษา 27 กรกฎาคม 2566

มาตรฐาน ผลิตภัณฑ์ อุตสาหกรรม เลขที่ มาตรฐาน รายละเอียด อัตราค่าใช้จ่าย ต่อตัวอย่าง 1 ชุด (บาท) หมายเหตุ แรงดึงขาด 420 แรงฉีกขาด 350 ความเรียบ ของผ้าภายหลัง การซักและ ทำให้แห้ง 3 รอบ ( Smoothness appearance ) 1,200 คุณลักษณะ ด้านความ ปลอดภัย ความเป็น กรด - ด่าง 430 ฟอร์มาลดีไฮด์ 1,350 อนุภาคโลหะหนัก - ตะกั่ว - แคดเมียม - โครเมียม ทั้งหมด - โครเมียม ( VI ) - ทองแดง 2,500 สี เอโซ ( azo dye ) ที่ให้แอโรแมติก แอมีน ( aromatic amine ) 3,500 การบรรจุ 100 เครื่องหมายและฉลาก 100 ทั้งนี้ ตั้งแต่วันประกาศในราชกิจจานุเบกษาเป็นต้นไป ประกาศ ณ วันที่ 6 มิถุนายน พ.ศ. 256 6 อนุชา นาคาศัย รัฐมนตรีประจำสำนักนายกรัฐมนตรี รักษาราชการแทน รัฐมนตรีว่าการกระทรวงอุตสาหกรรม ้ หนา 46 ่ เลม 140 ตอนพิเศษ 183 ง ราชกิจจานุเบกษา 27 กรกฎาคม 2566
