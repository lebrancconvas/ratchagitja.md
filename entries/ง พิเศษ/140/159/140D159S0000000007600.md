# ข้อบังคับของสำนักงานการบินพลเรือนแห่งประเทศไทย ฉบับที่ 26 ว่าด้วยใบรับรองผู้ดำเนินการเดินอากาศ (Air Operator Certificate)

ข้อบังคับของสํานักงานการบินพลเรือนแห่งประเทศไทย ฉบับที่ 26 ว่าด้วยใบรับรองผู้ดําเนินการเดินอากาศ (Air Operator Certi fi cate) อาศัยอํานาจตามความในมาตรา 6 / 1 มาตรา 15 ( 7 ) มาตรา 41 / 112 มาตรา 41 / 114 มาตรา 41 / 115 มาตรา 41 / 116 และมาตรา 41 / 118 แห่งพระราชบัญญัติการเดินอากาศ พ . ศ . 2497 ซึ่งแก้ไขเพิ่มเติมโดยพระราชบัญญัติการเดินอากาศ ( ฉบับที่ 14 ) พ . ศ . 2562 ประกอบกับ มติคณะกรรมการการบินพลเรือนในคราวประชุมครั้งที่ 3 / 2566 เมื่อวันที่ 7 มิถุนายน พ . ศ . 2566 ผู้อํานวยการสํานักงานการบินพลเรือนแห่งประเทศไทยจึงออกข้อบังคับเพื่อให้สอดคล้องตามภาคผนวก 6 และบทแก้ไขเพิ่มเติมภาคผนวก 6 แห่งอนุสัญญาว่าด้วยการบินพลเรือนระหว่างประเทศ ค . ศ . 1944 และ Commission Regulation (EU) 965/2012 Part-Organisation Requirements for Air Operations (Part-ORO) เกี่ยวกับคุณสมบัติของผู้มีสิทธิขอใบรับรองผู้ดําเนินการเดินอากาศ หลักเกณฑ์การขอ การออก การขอแก้ไขเปลี่ยนแปลง การต่ออายุ ใบรับรองผู้ดําเนินการเดินอากาศ เพื่อเป็นการรับรองว่าผู้ที่ยื่นขอรับใบรับรองดังกล่าวมีความสามารถดําเนินการเดินอากาศได้อย่างปลอดภัย และมีประสิทธิภาพไว้ ดังต่อไปนี้ ข้อ 1 ข้อบังคับนี้เรียกว่า “ ข้อบังคับของสํานักงานการบินพลเรือนแห่งประเทศไทย ฉบับที่ 26 ว่าด้วยใบรับรองผู้ดําเนินการเดินอากาศ (Air Operator Certificate)” ข้อ 2 ข้อบังคับนี้ให้ใช้บังคับตั้งแต่วันที่ 3 ธันวาคม พ . ศ . 2566 เป็นต้นไป ข้อ 3 ให้ยกเลิกข้อบังคับของคณะกรรมการการบินพลเรือน ฉบับที่ 8 5 ว่าด้วยใบรับรอง ผู้ดําเนินการเดินอากาศ ลงวันที่ 23 ธันวาคม พ . ศ . 2552 ข้อ 4 ในข้อบังคับนี้ “ ผู้อํานวยการ ” หมายความว่า ผู้อํานวยการสํานักงานการบินพลเรือนแห่งประเทศไทย “ สํานักงาน ” หมายความว่า สํานักงานการบินพลเรือนแห่งประเทศไทย ข้อ 5 ผู้มีสิทธิขอใบรับรองผู้ดําเนินการเดินอากาศต้องเป็นนิติบุคคลที่มีสัญชาติไทย และเป็นผู้ได้รับอนุญาตให้ประกอบกิจการขนส่งทางอากาศเพื่อการพาณิชย์ ในกรณีผู้มีสิทธิขอใบรับรองผู้ดําเนินการเดินอากาศได้ยื่นคําขออนุญาตให้ประกอบกิจการขนส่ง ทางอากาศเพื่อการพาณิชย์แล้ว และมีความประสงค์จะยื่นขอใบรับรองผู้ดําเนินการเดินอากาศ ผู้อํานวยการจะอนุญาตให้ยื่นคําขอใบรับรองผู้ดําเนินการเดินอากาศไปก่อนได้หากผู้ขอได้จัดทําระบบ บริหารจัดการ (Management System) ที่แสดงได้ว่ามีการเตรียมความพร้อมในการจัดให้มีมาตรฐาน การจัดการความปลอดภัยสําหรับการขนส่งทางอากาศเพื่อการพาณิชย์ โดยการจัดทําระบบบริหารจัดการ (Management System) ที่ประกอบด้วยระบบการจัดการด้านนิรภัย (Safety Management System) ้ หนา 76 ่ เลม 140 ตอนพิเศษ 159 ง ราชกิจจานุเบกษา 5 กรกฎาคม 2566

ระบบควบคุมติดตามและกํากับดูแลการดําเนินงานเพื่อให้เป็นไปตามกฎระเบียบ (Compliance and Monitoring System) หากปรากฏว่าผู้ขอไม่ได้รับอนุญาตให้ประกอบกิจการการบินพลเรือน ผู้อํานวยการจะคืนคําขอ ใบรับรองผู้ดําเนินการเดินอากาศพร้อมเอกสารหลักฐานที่เกี่ยวข้องให้กับผู้ขอ โดยจะไม่มีการคืน ค่าธรรมเนียมการยื่นคําขอและค่าใช้จ่ายในการดําเนินการที่มีการเรียกเก็บจากผู้ขอมาแล้ว ข้อ 6 ผู้ที่ประสงค์จะขอรับใบรับรองผู้ดําเนินการเดินอากาศให้ยื่นคําขอตามแบบที่ ผู้อํานวยการกําหนดล่วงหน้าก่อนวันที่คาดว่าจะเริ่มปฏิบัติการบินไม่น้อยกว่า 180 วัน พร้อมด้วย เอกสารและหลักฐานที่แสดงคุณสมบัติและลักษณะ ดังต่อไปนี้ ( 1 ) ชื่อ และที่อยู่ของผู้ขอ ( 2 ) สําเนาหนังสือรับรองหรือหลักฐานการเป็นนิติบุคคล ซึ่งแสดงรายการเกี่ยวกับชื่อ วัตถุประสงค์ ที่ตั้งสํานักงาน และผู้มีอํานาจลงนามผูกพันนิติบุคคลที่เป็นปัจจุบัน โดยมีคํารับรอง ของผู้มีอํานาจให้คํารับรองตามกฎหมายไม่เกินหก ( 6 ) เดือนนับแต่วันที่ออกหนังสือรับรองนั้น ( 3 ) สําเนาบัญชีรายชื่อผู้ถือหุ้นที่นายทะเบียนรับรอง ( 4 ) สําเนาใบอนุญาตประกอบกิจการการบินพลเรือน ( ถ้ามี ) ( 5 ) สถานที่ประกอบธุรกิจหลัก และสถานีหลักที่ใช้ปฏิบัติการ ( 6 ) โครงสร้างขององค์กรในเชิงธุรกิจ โดยระบุชื่อและที่อยู่ของผู้บริหารทุกตําแหน่ง รวมทั้ง ผู้มีอํานาจจัดการด้านการเงิน ( 7 ) ชื่อและที่อยู่ของผู้ที่ได้รับมอบอํานาจเพื่อดําเนินการยื่นคําขอและการอื่นที่เกี่ยวข้อง ( 8 ) สําเนาบัตรประจําตัวประชาชน หรือสําเนาหนังสือเดินทางของผู้ดํารงตําแหน่งบริหาร เช่น ประธานกรรมการ ผู้อํานวยการฝ่ายปฏิบัติการบิน ผู้อํานวยการฝ่ายช่าง ผู้อํานวยการ ฝ่ายฝึกอบรม หรือผู้อํานวยการฝ่ายปฏิบัติการภาคพื้น เป็นต้น ( 9 ) ลักษณะการปฏิบัติการบินที่เสนอขอ เช่น การบินขนส่งคนโดยสาร การบินขนส่งสินค้า การบินขนส่งไปรษณีย์ภัณฑ์ การปฏิบัติการบินกลางวันหรือกลางคืน การปฏิบัติการบินด้วยทัศนวิสัย หรือเครื่องวัดประกอบการบิน หรือการบินขนส่งสินค้าอันตราย ( ถ้ามี ) เป็นต้น ( 10 ) ข้อมูลเกี่ยวกับอากาศยาน ดังต่อไปนี้ ( ก ) แบบของอากาศยาน ( ข ) เครื่องหมายสัญชาติและทะเบียน ( ถ้ามี ) ( ค ) อุปกรณ์วิทยุสื่อสารและเครื่องช่วยในการเดินอากาศประจําอากาศยาน ( ง ) เอกสารที่เกี่ยวกับการซื้อขาย การเช่า (Dry Lease) และสัญญาหรือหนังสือแสดง เจตจํานงในการซื้อขายหรือการเช่า (Dry Lease) ( 11 ) การเตรียมการเกี่ยวกับการบํารุงรักษาอากาศยานและอุปกรณ์ที่เกี่ยวข้อง ้ หนา 77 ่ เลม 140 ตอนพิเศษ 159 ง ราชกิจจานุเบกษา 5 กรกฎาคม 2566

( 12 ) ข้อมูลเกี่ยวกับผู้ประจําหน้าที่ในอากาศยาน ใบอนุญาต และประสบการณ์การบิน ของแต่่ละคน ( 13 ) การกําหนดคุณสมบัติที่จะปฏิบัติงานของผู้ประจําหน้าที่ในอากาศยาน ลูกเรือ และ เจ้าหน้าที่ภาคพื้น แผนการจัดฝึกอบรม รวมทั้งวิธีการบันทึกประวัติการฝึกอบรมของบุคคลดังกล่าว ( 14 ) เส้นทางบิน ความถี่และประเภทของการบิน ( 15 ) แผนการอํานวยการบินและวิธีการกํากับดูแล ( 16 ) ข้อมูลทางด้านการเงิน เช่น ทุนและแหล่งเงินทุนตั้งต้นในการดําเนินการ บัญชีงบดุล ของปีที่ผ่านมา กระแสเงินสดที่เพียงพอต่อขนาดธุรกิจ เป็นต้น ( 17 ) คู่มือการปฏิบัติการซึ่งประกอบด้วยโครงสร้าง 4 หัวข้อหลัก ประกอบด้วย ( ก ) คู่มือการปฏิบัติการทั่วไป (Operations Manual Part A: General) ( ข ) คู่มือการปฏิบัติการเฉพาะแบบของอากาศยาน (Operations Manual Part B: Aircraft Operating information) ( ค ) คู่มือพื้นที่ปฏิบัติการบิน เส้นทางบิน และรายการสนามบิน (Operations Manual Part C: Areas, Route and Aerodrome) ( ง ) คู่มือหลักสูตรการฝึกอบรมพนักงาน (Operations Manual Part D: Training Programmes) ( 18 ) คู่มือการบํารุงรักษาทั่วไป (General Maintenance Manual) ( 19 ) แผนการบํารุงรักษาอากาศยาน (Aircraft Maintenance Programes) ( 20 ) เอกสารหลักฐานอื่นตามที่ผู้อํานวยการประกาศกําหนด คู่มือตาม ( 17 ) ให้มีรายละเอียดเป็นไปตามภาคผนวกแนบท้ายข้อบังคับนี้ กรณีผู้ขอใบรับรองผู้ดําเนินการเดินอากาศที่ใช้บัลลูนในการปฏิบัติการบินให้ยื่นคู่มือปฏิบัติการบินบัลลูน (Operations Manual - Balloon) แทนการยื่นคู่มือตาม ( 17 ) - ( 19 ) ข้อ 7 เมื่อผู้อํานวยการตรวจสอบคําขอและเอกสารหลักฐานประกอบคําขอตามข้อ 6 แล้ว มีความครบถ้วนถูกต้อง ให้ดําเนินการตามขั้นตอน ดังต่อไปนี้ (1) ขั้นตอนการเตรียมการในเบื้องต้นของผู้ขอรับใบรับรองผู้ดําเนินการเดินอากาศ (Pre - Application Phase) ( 2 ) ขั้นตอนการยื่นคําขออย่างเป็นทางการ (Formal Application Phase) ( 3 ) ขั้นตอนการตรวจสอบเอกสารหลักฐาน (Document Evaluation Phase) ( 4 ) ขั้นตอนการตรวจสอบการปฏิบัติการ (Demonstration and Inspection Phase) ( 5 ) ขั้นตอนการออกใบรับรอง (Certification Phase) ข้อ 8 ผู้อํานวยการจะดําเนินการออกใบรับรองผู้ดําเนินการเดินอากาศและข้อกําหนด การปฏิบัติการ (Operations Specifications) พร้อมทั้งเอกสารแนบท้ายใบรับรองผู้ดําเนินการ ้ หนา 78 ่ เลม 140 ตอนพิเศษ 159 ง ราชกิจจานุเบกษา 5 กรกฎาคม 2566

เดินอากาศ (Attachment to Operations Specifications) เมื่อได้ดําเนินการครบถ้วนทั้ง 5 ขั้นตอน ตามข้อ 7 และได้ประเมินความพร้อมทางด้านองค์กร มาตรการควบคุมและการกํากับดูแล การปฏิบัติการบิน แผนการจัดฝึกอบรม แผนรักษาความปลอดภัย การจัดการซ่อมบํารุง การสาธิต การปฏิบัติการแล้ว ปรากฏว่ามีความสอดคล้องกับลักษณะของการปฏิบัติการที่เสนอขอ และสามารถ
